#Giant Machines Work-Order Back-End

This application is built with Ruby on Rails to organize and sort work orders

## Installation

Once cloned run…

```
bundle install

rails db:create
rails  db:migrate
rails db:seed

rails s -p 8000
```
This should have the application running on localhost:8000
